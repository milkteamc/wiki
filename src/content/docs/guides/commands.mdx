## 指令使用提示

:::tip[參數說明]
- `[玩家名稱]`：輸入其他玩家的遊戲名稱
- `[金額]`：輸入數字，例如 100、1000
- `[名稱]`：輸入家點名稱，例如 home1、基地、農場
- `[密碼]`：輸入你的帳號密碼
:::
