## 連線相關

### 無法連線到伺服器怎麼辦？

1. 確認你的 Minecraft 版本在支援範圍內
2. 嘗試使用備用連線位址（僅限 Java 版本）
3. 檢查你的網路連線是否穩定
4. 如果問題持續，請到 [Discord](https://discord.gg/22DxRjrXRv) 社群尋求幫助

### 基岩版連線不穩定怎麼辦？

基岩版的連線確實比較不穩定，你可以：
- 嘗試重新連線
- 嘗試使用好友邀請的方式加入

## 帳號相關

### 忘記密碼了怎麼辦？

請到我們的 [Discord](https://discord.gg/22DxRjrXRv) 社群開啟客服單，管理員會協助你重設密碼。

### 可以更改用戶名嗎？

遊戲內的用戶名無法更改，如果進行更改，原有物品皆會遺失，需在 Discord 社群中開啟客服單詢問。

## 技術問題

### 材質包無法下載？

1. 確認你的網路連線穩定
2. 嘗試手動輸入 `/download` 指令
3. 如果問題持續，請聯絡管理員
